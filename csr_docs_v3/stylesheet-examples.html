Headers

Header 1

Header 1 is made by selecting the 'H1' formatting selector or nesting text inside a <h1> tag. This header is always automatically centered.

Header 2

Header 2 is made by selecting the 'H2' formatting selector or nesting text inside a <h2> tag

Header 3

Header 3 is made by selecting the 'H3' formatting selector or nesting text inside a <h3> tag

Header 4

Header 4 is made by selecting the 'H4' formatting selector or nesting text inside a <h4> tag

Header 4 (Small)

Header 4 (small) is made by selecting the 'H4' formatting selector or nesting text inside a <h4> tag and selecting the ".rs-h4-small" class
Header 5
Header 5 is made by selecting the 'H5' formatting selector or nesting text inside a <h5> tag
Header 5 (address)
Header 5 (address) is made by selecting the 'H5' formatting selector or nesting text inside a <h5> tag and selecting the ".address" class

Buttons

Buttons are formed by creating a link (using the <a> tag, properly formed) and assigning it the class ".btn" and a second class for the specific type of button.

To create this button...
...create a button with this class...
...and tell it the type of button with this class.
Default
.btn
.btn-default
Primary
.btn
.btn-primary
Secondary
.btn
.btn-secondary
Tertiary
.btn
.btn-tertiary
As a link
This is a standard hyperlink. It does not require a .btn class of any kind.

These are innactive or "dead" buttons, use these when an option would normally be available but for purposes of the user's current context is otherwise unreachable or not applicable. Similar to the buttons above, these are created by adding all the previous classes to define the button and the ".btn-off-*" class as appropriate for the style of button.
